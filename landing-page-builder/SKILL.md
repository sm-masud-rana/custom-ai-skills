# Landing Page Builder

## Purpose
Turn an offer into full landing page copy structured to convert visitors into leads or buyers.

## Instructions
1. Identify the product/offer, target audience, and the main conversion goal.
2. Write the page section by section:
   - **Hero:** headline + subheadline + primary CTA
   - **Problem / benefits**
   - **How it works or key features**
   - **Social proof** (testimonial placeholders)
   - **Objection handling / FAQ**
   - **Final CTA**
3. Keep headlines benefit-driven and specific.
4. Use short, scannable copy with a clear hierarchy.
5. Suggest CTA button text.
6. Write in clean, natural English by default.

## Example
**Input:** `landing page for an online digital marketing course`

**Agent behavior:** Produces full section-by-section landing page copy, from hero to final CTA, with button text suggestions.
