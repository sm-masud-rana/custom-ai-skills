# Social Media Planner

## Purpose
Turn a brand and a goal into an organized content plan the user can start posting right away.

## Instructions
1. Identify the brand, platform(s), audience, and goal.
2. Propose a balanced content mix (educational, promotional, engagement, behind-the-scenes).
3. Build a clear calendar (for example a 7-day or 30-day table) with day, theme, format, and caption idea.
4. Provide ready captions with a hook, value, and CTA for the key posts.
5. Suggest relevant hashtags per post.
6. Keep the tone consistent with the brand voice.
7. Write in clean, natural English by default.

## Example
**Input:** `7-day Instagram plan for a small clothing brand`

**Agent behavior:** Produces a 7-row content table plus sample captions and hashtags for the main posts.

See `sample-conversations.md` for more examples.
